## 1. 世界の根源原理

### 原虹 (げんそう)
- 万物の源
- 7色の光 (赤橙黄緑青藍紫)
- すべての生命は「**原虹の一筋の光 = 虹脈 (こうみゃく)**」を持つ

### 観測者の七座
原虹の各色に対応する原理の人格化。主席プリズマ + 7座 = 計8存在。

| 色 | 座 | 担当原理 | 現任 (S1) |
|---|----|---------|---|
| 🔴 赤 | 第一席 | **祈り** | セラフィエル |
| 🟠 橙 | 第二席 | **記憶** | カグヤ |
| 🟡 黄 | 第三席 | **未来** | ノクス |
| 🟢 緑 | 第四席 | **約束** | 空席 → S1C6 リーリエ着任 |
| 🔵 青 | 第五席 | **連結** | 空席 → S2 |
| 🟣 藍 | 第六席 | **忘却** | 空席 → S2 |
| 💜 紫 | 第七席 | **沈黙** | 空席 → S3 |
| 🌈 主席 | プリズマ | 統べる虹そのもの | 主席 (S1終盤に眠る) |

### 観測者の三柱 (みはしら)
セラフィエル・カグヤ・ノクスの呼称。**姉妹ではなく、それぞれ独立した領土と気質**。普段は会わず、危機時のみ集結。

- **セラフィエル** (祈り): 真面目で直球、新参の聖騎士気質。地上に頻繁に降りる
- **カグヤ** (記憶): 千年見てきた飄々とした賢者。月から滅多に動かず、助言だけ残して消える
- **ノクス** (未来): 全てを知って沈黙する未来予知者。何を考えてるか不明、宇宙の彼方

### 影喰いと裂け目
- **影喰い** = 観測されない側の意志
- **裂け目** = 世界のほつれ、影喰いの侵入路
- 最大の敵: **無虹の王 ヴォイドラ** (S1C7 覚醒)

## 3. Season構造 (大紀)

### 第一大紀 (Season 1) 「観測の時代」
七座が世界を維持。古い秩序の章。プリズマ眠り入りで終わる。

### 第二大紀 (Season 2) 「接続の時代」
プリズマ眠り、並行世界と接続。Season1キャラの並行同位体出現。

### 第三大紀 (Season 3) 「超越の時代」
二元論を超えた「**共観 (きょうかん)**」へ。世界新生。

### 第四大紀以降: 未定 (永劫回帰の構造を想定)

- 第1章: 序 (大規模)
- 第2章: 拡張 (中規模)
- 第3章: 異変 (小規模)

## 5. キャラ管理

### ID体系
`S1C1-LR-01` 形式 (Season-Chapter-Tier-番号)

### カテゴリ
| ID | 用途 | 公式canon |
|----|----|----|
| `main` | 公式メイン | ✅ |
| `past` | 過去編 (公式準拠) | ✅ |
| `if` | IFスピンオフ | ❌ (逸脱許容) |
| `event` | イベント別衣装 (期間限定→恒常入り) | ✅ |
| `spinoff` | サブストーリー専属 | △ |

### 凸システム (設定解放型)

被りで「**秘話/裏設定**」が1段ずつ解放される。

| tier | 段数 | 解放内容 |
|----|----|----|
| **R** | 2段 | Lv1=公開desc / Lv2=隠しエピ |
| **SR** | 3段 | + Lv3=過去エピ |
| **SSR** | 4段 | + Lv4=他キャラとの裏設定 |
| **UR** | 5段 | + Lv5=Season跨ぎ伏線 |
| **LR** | 5段 + 演出強化 | 各段で背景演出upgrade |

凸演出: 「○凸目」テキスト表示のみ (簡素)

### 関係性タイプ

| Season | タイプ | 用途 |
|----|----|----|
| **S1** | fellow / master / blood / childhood / admire / rival / sister / oath | 基本関係 |
| **S2 追加** | parallel (並行同位体) / descendant (子孫) / mentor-from-future (未来師) | 並行世界 |
| **S3 追加** | transcend (時空超越) / sevencolor (七色使徒同志) | 超越関係 |

## 7. サイドコンテンツ

### イベント
- 別衣装Ver
- **3-4章に1回**のペース (キャラ増えたらスパン伸ばす)
- 期間限定→恒常入り (限定キャラなし)

### スピンオフ (IF)
- 「もし○○だったら」
- 公式設定からの逸脱許容
- 例: 暗黒IFアルテミス、戦巫女リナエ

### 過去編
- 既存キャラの若年版/前史
- **公式設定準拠**
- 例: 百年前の観測者、少年時代の黒刃玄、若き日のヒノオウ

### サブ章 (世界観深堀り)
- 本筋に直接絡まない
- 新キャラ無しでもOK

## 8. ストーリー記述

### ボリューム
| 章規模 | 推奨字数 | 文庫換算 |
|----|----|----|
| **大章** (S1C1/S1C7) | **1-2万字** | 16-33ページ |
| **中章** (S1C4/S1C6) | **1万字前後** | 16ページ |
| **小章** (S1C3/S1C5) | **5,000-8,000字** | 8-13ページ |

### 形式
- **紙芝居風UI**: 背景画像 + キャラ立ち絵 (既存カード絵流用) + テキスト枠
- **表情差分なし**
- 執筆スタイル: **ハイブリッド** (小説風 + シナリオ風)
- Claude が下書き、野沢さんは推敲なし、相談ベース

### 配置
- ホーム画面に「📖 ストーリー」セクション
- キャラ詳細画面からも該当ストーリーへ導線

### 執筆フロー
- **第1章は章単位で全部書く** (1-2万字を一気に)
- **第2章以降は都度リクエスト方式** (シーン単位や章単位で気が向いた時に)
